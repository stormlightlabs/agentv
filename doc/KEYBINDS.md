# Keyboard Shortcuts

This document lists all available keyboard shortcuts in the Agent V desktop application.

## Global Shortcuts

These shortcuts are available from any screen in the application.

| Shortcut | Action |
| :--- | :--- |
| `Cmd + K` | Open Command Palette / Focus Search |
| `Cmd + R` | Refresh all sessions |
| `Cmd + B` | Toggle Bookmarks pane |
| `Cmd + 1` | Switch to **Sessions** tab |
| `Cmd + 2` | Switch to **Search** tab |
| `Cmd + 3` | Switch to **Analytics** tab |
| `Cmd + 4` | Switch to **Status** tab |
| `Cmd + D` | Bookmark current session (requires active session) |

## Navigation & UI

### Sidebar Resizing

Focus the resizer handle between the sidebar and main content to use these shortcuts.

| Shortcut | Action |
| :--- | :--- |
| `Arrow Left` | Shrink sidebar width |
| `Arrow Right` | Expand sidebar width |
| `Home` | Set sidebar to minimum width |
| `End` | Set sidebar to maximum width |

### Command Palette

Active when the Command Palette is open.

| Shortcut | Action |
| :--- | :--- |
| `Arrow Up / Down` | Navigate through commands |
| `Enter` | Execute selected command |
| `Escape` | Close the Command Palette |

## Contextual Shortcuts

### Filter Bar / Search

Active when the search input or filter bar is focused.

| Shortcut | Action |
| :--- | :--- |
| `Enter` | Trigger search with current filters |
| `Escape` | Close the filters dropdown |
| `Cmd + K` | Quickly refocus the search input |

### Session Viewer

Active when navigating events within a session.

| Shortcut | Action |
| :--- | :--- |
| `Enter / Space` | Toggle event expansion (show thinking/raw data) |
| `i` | Open event in the detailed Inspector |
